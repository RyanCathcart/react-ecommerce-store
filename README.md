# React E-Commerce Store
A basic example of an e-commerce store application developed using .NET and TypeScript that can be used to simulate purchasing various (fake) merchandise. Features a login and authentication system, and a user shopping cart.

## Technologies Used:
#### Back End: <span>ASP.NET</span> Core API
- Language: C# 11.0
- Frameworks/Packages/Technologies:
  - PostgreSQL 7.0.1 - Database
  - SQLite 7.0.1 - Database
  - Entity Framework Core 7.0.1 - Object-Relational Mapper
    
#### Front End: React
- Language: TypeScript 4.4.3
- Frameworks/Packages/Technologies:
  - Axios
  - Redux
  - Material UI
  - React-hook-form
  - React-Router
